Place your location photos in this folder.

To add a photo to a location:
1. Add the photo file to this folder (e.g., "paris.jpg")
2. In travel_data.geojson, add a "Photo" property to the location's properties:
   "Photo": "paris.jpg"

The photo will automatically appear in the popup when you click on the location marker.

Supported formats: JPG, PNG, GIF, WebP
